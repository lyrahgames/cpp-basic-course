What is C++?

C++ is a general-purpose programming language emphasizing the design and use of type-rich, lightweight abstractions. It is particularly suited for resource-constrained applications, such as those found in software infrastructures. C++ rewards the programmer who take the time to master techniques for writing quality code. C++ is a language for someone who takes the task of programming seriously.
The C++ Programming Language, 4. Edition by Bjarne Stroustrup
